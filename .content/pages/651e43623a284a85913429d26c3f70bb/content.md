[MOHARA ](https://mohara.co/)rolled out Github Copilot a little while back, and recently [Copilot Chat](https://docs.github.com/en/copilot/github-copilot-chat/using-github-copilot-chat) come out into public beta. 
<empty-block/>
I’m just scratching the surface of ways to use it, but here was something that I found helpful:
1. Highlight the code in question
2. Right click and select, `Copilot` then `Start code chat`
3. Write a prompt in there 
	![](/notion-images/cccab07a4e428f1d.png)
4. It will then give you a diff to show the changes that it suggests and you can accept or reject it right there in the file 🎉
	![](/notion-images/c72bad4d50775be6.png)
